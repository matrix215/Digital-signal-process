# 실험 목적
MATLAB을 이용하여 원본 사진을 DFT한 값과 Inverse DFT한 값의 결과를 비교 분석. 

# 실험 내용
원본 이미지를 DFT한 후의 결과와, 다시 복원한 IDFT를 한 후의 결과를 관찰하고, MSE값과 PSNR의 값을 구해서 얼마나 비슷하게 복원되었는지를 확인할 수 있다.
